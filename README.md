# ☕ Coffee Mais

Projeto em desenvolvimento.

## Stack

- **Backend:** [Supabase](https://supabase.com) (PostgreSQL, Auth, Storage, Edge Functions)
- **Metodologia:** [BMAD Method](https://docs.bmad-method.org/) v6

## Setup

### 1. Clonar o repositório

```bash
git clone <url-do-repo>
cd coffe-mais
```

### 2. Instalar dependências

```bash
npm install
```

### 3. Configurar variáveis de ambiente

```bash
cp .env.example .env.local
# Edite .env.local com suas credenciais
```

### 4. Rodar o projeto

```bash
npm run dev
```

## BMAD Method

Este projeto usa o BMAD Method para desenvolvimento estruturado com IA.

```bash
# Orientação de workflow
/bmad-help

# Agentes disponíveis
/pm          # Product Manager
/architect   # Arquiteto
/dev         # Desenvolvedor Senior
/qa          # Quality Assurance
```
